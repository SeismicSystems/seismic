# TypeScript

Seismic's TypeScript SDKs for building dApps on Seismic.

* **seismic-viem** — Core client library built on [viem](https://viem.sh/) for interacting with a Seismic RPC provider
* **seismic-react** — React hooks and providers built on seismic-viem and [wagmi](https://wagmi.sh/)
